---

## 🌟 Features

- 🎨 **Responsive Design** — Powered by Bootstrap's grid system, looks good on desktop, tablet, and mobile
- 🧩 **Project Showcase** — Section highlighting your best work
- 📄 **Resume Section** — Easy access to your CV
- 📬 **Contact Section** — Ways for visitors to reach out
- 🔗 **Social Links** — Quick access to LinkedIn, GitHub, etc.
- 🪶 **Lightweight** — No JavaScript or heavy dependencies, just fast-loading static pages

---

## 🚀 Getting Started

Since this is a static site with no build process, you don't need Node.js, npm, or any package manager.

## 🧭 Sections

| Section | Description |
|---|---|
| **Home** | Intro with your name, title, and a short tagline |
| **About** | Your background, skills, and story |
| **Projects** | Showcase of your best work with live links & repos |
| **Skills** | Technologies and tools you're familiar with |
| **Contact** | Contact details / social links |
